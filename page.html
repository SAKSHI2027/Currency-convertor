This is my Currency Converter application

Currency conversion is the process of changing one
currency into the currency of another country.
For example, changing Australian dollars (AUD) into US Dollars (USD).
Currencies are converted using an exchange rate.

A currency converter app is a mobile application
that allows users to convert one currency to
another using the latest exchange rates.
These apps are particularly useful for
travelers, businesses, and anyone dealing with multiple currencies.
